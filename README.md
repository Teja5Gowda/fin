This is a personal finance tracker for students. 

Features:
1. User Authentication: Secure registration and login functionality.
2. Transaction Management: Add, delete, and view expense and income transactions.
3. Analytics: Insights into spending patterns by payment method and category.
4. Responsive Interface: User-friendly design for easy navigation and interaction.

Installation:
1. Clone the repository
2. cd fin
3. pip install -r requirements.txt
4. python app.py
